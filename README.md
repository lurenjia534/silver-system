# Spring boot 项目，使用了微软 OAuth2.0 登录，使用了微软 Graph API 获取用户信息，使用了微软 OneDrive API。

## 项目结构

```
src
├── main
│     ├── java
│         └── com.example.demo
│           ├── AppConfig.java # 配置类
│           ├── controller.java # 控制器
│           ├── DemoApplication.java # 启动类
│           ├── TokenService.java # Token 服务

```
